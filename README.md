# UpRollView
广告条 ， 文字上下滚动轮播
四种方式实现文字或View上下滚动轮播，仿淘宝头条、淘宝广告条；
- TextView+Handler延时动画 实现滚动效果；
- 自定义ViewFlipper 实现滚动效果；
- RecyclerView+子线程延时 实现滚动效果；
- 竖直ViewPager+Handler延时 实现滚动效果；
